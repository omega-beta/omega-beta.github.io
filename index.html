Welcome to BLOCK BLASTER!

HOW TO PLAY

Navigation is with the standard WASD (W: foward, A: left, S: back, and D: right), and the rest is basic. Look around using your mouse. Fire by clicking, and your selected gun will fire in the direction of your mouse.

You may also access our game by the header with 'Gameplay'.

HOW TO USE

  1. Install the Emscripten SDK and follow this.
  2. Type git clone https://github.com/ichang3765-alt/blockblaster.git && cd blockblaster to your bash terminal/Linux.
  3. Build index.js and index.wasm with emcc main.cpp -s WASM=1 -s USE_SDL=2 -03 -o index.js.
  4. Lastly, Chrome and other web browsers don't support file://XHR requests, so start a webserver like so: python3 -m http.server 8080.
  5. Open http://localhost:8080/.

That's it for setting up!

If you want more info, just right click and press "Inspect". Then, access the README.md file. If you want more like this, check out Taming.io and Voxiom.io

RULES & ADVANCED PLAY

Remember, have good sportsmanship, and be modest.

  1. Each gun has a different amount of damage. Experiment with the guns to get the best start.
  2. You will start out with a knife at the same time as other people on the max 15-person Battle Royale. Deployment will be at the designated time.
  3. As you upgrade, the guns will get better.
  4. If you get killed, don't worry. It's just a game.
  5. Remember that this place has boundaries. You can use them as walls.
  6. There are hiding spots. Use them to your advantage.
  7. Please note that this is only the Alpha phase, version 1.0.0.
